## 📖 Acerca de BlockBookster

BlockBookster es una **plataforma social de lectura** donde los usuarios pueden:

- Registrar y calificar libros.
- Crear y compartir listas de lectura.
- Escribir y leer reseñas.
- Seguir a otros usuarios y autores.
- Recibir recomendaciones personalizadas.
- Acceder a enlaces de compra con sistema de afiliación.

**Tecnologías principales:**

- Backend: Laravel 12, Eloquent ORM, Sanctum
- Frontend: Blade + TailwindCSS
- Base de datos: MySQL / PostgreSQL
- Autenticación: Token-based (roles: user, worker, admin)
- Testing: PHPUnit / Pest
- Diseño: Neo-brutalism + UI/UX responsive

## 🚀 Características

### Libros
- Búsqueda avanzada y filtros exhaustivos
- Rating medio, popularidad y tendencias
- Información completa de autores
- Enlaces de compra con afiliación

### Autores
- Biografía, valoraciones y seguidores
- Relación con sus libros y coautores

### Usuarios
- Registro, login y perfil público
- Seguimiento de listas y reseñas
- Gestión de favoritos y feeds

### Listas
- Colecciones curadas y ordenables
- Visibilidad: pública, privada, amigos

### Social
- Likes, follows, recomendaciones y feed personalizado
- Restricciones: un like/review por usuario por libro

### Administración
- Panel de administración para moderación
- Gestión de usuarios, libros, reviews y listas
